PyCharm 2019.3.1 x64安装以及激活教程

发表于 2019-12-28 更新于 2020-07-15 分类于 开发工具

说明

  • 经过测试发现：本教程适用于2019.3.1版本。
  • 本方法同样适用于其他 Jetbrains 产品，如 IntelliJ IDEA 2019.2.4 x64。

准备

  1. 下载 PyCharm 2019.3.1 x64
  2. 下载 补丁文件

安装软件

安装好下载的PyCharm 2019.3.1 x64。

激活&破解

  1. 将补丁文件jetbrains-agent.jar放入安装目录D:\Program Files\JetBrains\PyCharm 2019.3.1\bin。
  2. 打开Pycharm，此时提醒是否导入配置文件。如果你有配置文件，就可以选择导入。也可以选择不导入任何配置文件。这里先不导入配置文件。
  3. 选择完配置文件后就转到激活的地方，这里先选择Evaluate for Free>Evaluate>Continue。
  4. 点击Configure>Edit Custom VM Options。在文件的最后一行添加-javaagent:D:\Program Files\JetBrains\PyCharm 2019.3.1\bin\jetbrains-agent.jar
  5. 重启Pycharm。打开或者新创建一个项目，之后点击Help>Register，选择Activation code方式激活。激活从以下三个任选其一即可。

如果是导入了配置文件则在第4步用以下方法：
4. 修改C:\Users\hervie\.PyCharm2019.3\config\pycharm64.exe.vmoptions。根据你自己的用户名去寻找到pycharm64.exe.vmoptions这个文件。在文件的最后添加一行-javaagent:D:\Program Files\JetBrains\PyCharm 2019.3.1\bin\jetbrains-agent.jar

以下激活码任选其一即可
